# FanControl

Project for fan control of up to two 4 pin pwm fans (12V) with rpm validation controlled by DHT11 temperature and humidty sensor. Support for Arduino Nano (ATMega328P) and ESP8266 (NodeMCU) with JSON WebAPI.
